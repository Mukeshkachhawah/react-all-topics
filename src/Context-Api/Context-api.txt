Context API Kya Hai?
Context API ek aisa feature hai jo React mein props drilling se bachne ke liye use hota hai. Ye data ko directly parent se kisi bhi child component mein share karne ki capability deta hai, chahe wo kitne bhi levels deep ho.


Kab Use Karein (When to Use):

1. Global Data Sharing:

Theme (dark/light mode)
User authentication data
Language preferences
Shopping cart data
